## Getting Started

### Prerequisites

-   Bun [Install Instructions](https://bun.sh/docs/installation)

### Installation

1. Clone the repo

    ```sh
    git clone git@github.com:sparrowstack/sparrowstack.git
    ```

2. Install packages
    ```sh
    bun install
    ```
3. Build the project
    ```sh
    bun build:all
    ```
